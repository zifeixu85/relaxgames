How to Play

Candy Bubble is a fun match-3 puzzle game that's easy to learn but challenging to master!

  1. Swap adjacent candies to create matches of three or more of the same color
  2. Create special candies by matching four or more in a row
  3. Trigger special effects by combining power-ups
  4. Complete level objectives before running out of moves
  5. Progress through increasingly challenging levels with different goals

Use strategy and quick thinking to solve each sweet puzzle and advance to the next level!
